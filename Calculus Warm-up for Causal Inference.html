DAY 1 — DERIVATIVES

What is a derivative?

A derivative measures how much a function changes when its input changes. Geometrically: it's the slope of the curve at a point.

f'(x) = lim[h→0] (f(x+h) - f(x)) / h
⟶ Why this matters for causal inference: later you'll maximize likelihood functions to estimate treatment effects. Maximizing = finding where the derivative = 0.

The power rule (the one you'll use 90% of the time): if f(x) = xⁿ, then f'(x) = n·xⁿ⁻¹

[1]

The chain rule

When functions are nested — like f(g(x)) — you use the chain rule: (f∘g)'(x) = f'(g(x)) · g'(x)

d/dx [f(g(x))] = f'(g(x)) · g'(x)
⟶ Why this matters: backpropagation in neural networks (your DL track) is entirely the chain rule applied recursively. Also used when differentiating log-likelihoods.
💡 Example: d/dx[e^(x²)] = e^(x²) · 2x — outer derivative times inner derivative

Plotting derivatives — seeing the slope

The best way to build intuition is to see the function and its derivative side by side.

💡 Where f is increasing → f' is positive. Where f peaks → f' = 0. Where f is decreasing → f' is negative.

DAY 2 — INTEGRALS

What is a definite integral?

A definite integral computes the area under a curve between two points. It's the reverse of differentiation.

∫ₐᵇ f(x) dx = F(b) - F(a) where F'(x) = f(x)
⟶ Why this matters for probability: For a continuous random variable X with density f(x), the probability P(a ≤ X ≤ b) = ∫ₐᵇ f(x)dx. You will write this every single week in Blitzstein.

Basic rule: ∫ xⁿ dx = xⁿ⁺¹/(n+1) + C
